# Getting Started

In order to get started with byorg you don't need any huge setup. All you need is an LLM provider of your choice.

## Pre-requisites

- API keys to LLM Provider supported by Vercel's AI SDK (e.g. OpenAI, Anthropic, ...) [complete list here](https://sdk.vercel.ai/providers/ai-sdk-providers).

:::tip

You can even use self hosted LLM as long as it works with Vercel AI SDK

:::

## Installation

Easiest way to start is to simply install byorg using your package manager.

<PackageManagerTabs command="install @callstack/byorg-core ai" />

In next section we'll show you how to use our core library to handle your requests.
